# 用例管理界面使用指南

## 📐 界面布局

### 左侧：项目结构树（可折叠）
- **折叠/展开按钮**：点击左上角 `‹` / `›` 按钮
- **新建项目按钮**：快速创建新项目
- **刷新按钮**：重新加载项目数据
- **搜索框**：搜索项目/模块/接口/用例

### 右侧：详情展示区
根据左侧选中的层级显示不同内容：

#### 1. 项目层级 📁
显示内容：
- 项目统计卡片（模块数、用例总数、通过/失败/未执行）
- 模块列表表格
- 通过率进度条
- 最后执行时间

操作：
- 编辑项目
- 删除项目
- 新建模块
- 点击模块进入模块详情

#### 2. 模块层级 📂
显示内容：
- 模块统计卡片（接口数、用例总数、通过/失败/未执行）
- 接口列表表格
- 通过率进度条
- 最后执行时间

操作：
- 编辑模块
- 删除模块
- 新建接口
- 点击接口进入接口详情

#### 3. 接口层级 🔌
显示内容（多标签页）：
- **基本信息**：所属项目、模块、接口名称、路径、方法、描述、标签、前置条件
- **请求参数**：JSON格式的请求参数示例（可复制）
- **响应结果**：预期结果 vs 实际结果对比
- **测试历史**：历史执行记录列表
- **相关用例**：该接口下的所有测试用例

操作：
- 保存修改
- 执行测试
- 删除接口
- 点击用例进入用例详情

#### 4. 用例层级 📄
显示内容：
- 用例基本信息
- 请求参数（JSON格式）
- 预期结果
- 实际结果（失败时）
- 执行历史时间轴
- 相关用例列表

操作：
- 关闭详情面板
- 复制参数

## 🎯 操作流程

### 查看项目统计
1. 在左侧树中点击项目节点
2. 右侧显示项目统计信息
3. 查看模块列表和通过率

### 查看接口详情
1. 展开项目 → 展开模块 → 点击接口
2. 右侧显示接口详情的多个标签页
3. 切换标签页查看不同信息
4. 可以编辑接口信息并保存

### 查看用例详情
1. 展开到接口层级
2. 点击接口下的用例
3. 右侧显示用例详细信息
4. 查看执行历史和测试结果

### 新建内容
- **新建项目**：点击左侧工具栏的"新建项目"按钮
- **新建模块**：在项目节点上点击 ➕ 按钮
- **新建接口**：在模块节点上点击 ➕ 按钮
- **新建用例**：在接口节点上点击 ➕ 按钮

### 编辑和删除
- 每个节点都有 ✎（编辑）和 🗑（删除）按钮
- 悬停时显示操作按钮
- 删除前会弹出确认对话框

## 🎨 UI 特性

### 层级颜色区分
- **项目**：淡蓝色渐变背景
- **模块**：灰色渐变背景
- **接口**：白色背景
- **用例**：卡片式设计

### 状态指示
- **通过** ✓：绿色
- **失败** ✗：红色
- **未执行** ⊙：灰色

### 请求方法标签
- **GET**：绿色
- **POST**：蓝色
- **PUT**：橙色
- **DELETE**：红色
- **PATCH**：灰色

### 交互效果
- 悬停高亮
- 展开/收起动画
- 侧边栏折叠动画
- 标签页切换动画

## 🔄 数据流转

```
选择项目
  ↓
显示项目统计 + 模块列表
  ↓
点击模块
  ↓
显示模块统计 + 接口列表
  ↓
点击接口
  ↓
显示接口详情（5个标签页）
  ↓
点击用例
  ↓
显示用例详情（可关闭）
```

## 🚀 快捷操作

- **Ctrl + F**：聚焦搜索框
- **Esc**：关闭当前对话框
- **左侧折叠**：节省空间，专注内容

## 📝 注意事项

1. 侧边栏折叠后只显示折叠按钮
2. 用例详情可以随时关闭，不影响其他层级
3. 接口详情的标签页状态会保持
4. 搜索会高亮匹配的内容
5. 通过率自动计算，实时更新

## 💡 提示

- 频繁使用的功能放在了显眼位置
- 重要操作都有二次确认
- 统计数据实时更新
- 支持键盘快捷键
- 响应式设计，适配不同屏幕
