# 报告中心页面说明

## 概述
报告中心页面用于查看和管理测试报告，提供测试结果的可视化展示和报告导出功能。

## 功能特性

### 1. 报告列表展示
- **报告ID**：唯一标识每个测试报告
- **关联项目**：显示测试报告所属的项目
- **测试时间**：记录测试执行的时间
- **执行耗时**：显示测试执行所需的时间
- **通过率**：
  - 使用进度条可视化展示
  - 颜色编码：
    - 绿色：通过率 ≥ 90%
    - 橙色：70% ≤ 通过率 < 90%
    - 红色：通过率 < 70%
- **状态**：显示测试结果（通过/失败）
- **操作**：查看详情、下载PDF

### 2. 筛选功能
- **项目筛选**：按项目过滤报告
- **日期范围**：选择开始和结束日期
- **状态筛选**：按测试状态（通过/失败）过滤
- 所有筛选条件可组合使用

### 3. 导出功能
- 支持导出当前筛选条件下的报告
- 点击"导出报告"按钮触发导出

### 4. 报告操作
- **查看详情**：查看单个报告的详细信息
- **下载PDF**：下载报告的PDF版本

### 5. 分页功能
- 支持自定义每页显示数量（10/20/50/100条）
- 显示总记录数和当前显示范围
- 页码导航

## 技术实现

### 组件结构
```
Reports.vue
├── 面包屑导航
├── 页面标题
├── 筛选区域
│   ├── 项目选择
│   ├── 日期范围
│   ├── 状态筛选
│   └── 导出按钮
├── 报告表格
└── 分页组件
```

### API接口
位置：`src/api/report.js`

- `getReportList(params)` - 获取报告列表
- `getReportDetail(id)` - 获取报告详情
- `exportReport(params)` - 导出报告
- `downloadReportPDF(id)` - 下载PDF
- `deleteReport(id)` - 删除报告
- `getReportStatistics(params)` - 获取统计数据

### 数据结构
```javascript
{
  id: 'RPT-20240310-001',        // 报告ID
  projectName: '电商支付系统',    // 项目名称
  testTime: '2024-03-10 14:30',  // 测试时间
  duration: '2分钟',              // 执行耗时
  passRate: 95,                   // 通过率（百分比）
  status: 'passed'                // 状态：'passed' | 'failed'
}
```

## 使用方法

### 访问页面
导航至：`/reports` 或点击左侧菜单"报告中心"

### 筛选报告
1. 选择项目（可选）
2. 选择日期范围（可选）
3. 选择状态（可选）
4. 系统自动根据筛选条件刷新列表

### 导出报告
1. 设置筛选条件
2. 点击"导出报告"按钮
3. 确认导出
4. 等待导出完成

### 查看详情
1. 找到目标报告
2. 点击"查看详情"按钮
3. 在详情页面查看完整信息

### 下载PDF
1. 找到目标报告
2. 点击"下载PDF"按钮
3. 等待下载完成

## 开发说明

### 模拟数据
在API接口未完成时，页面会自动使用模拟数据进行展示，包含10条示例报告记录。

## 注意事项

1. **权限控制**：确保用户有查看报告的权限
2. **数据安全**：敏感报告数据需要加密传输
3. **性能优化**：大量报告时使用虚拟滚动
4. **错误处理**：网络错误时友好提示
5. **导出限制**：限制单次导出的报告数量
